# 🩺 Doctor Search Website

## 🔗 Live Demo
http://doctor-search.gt.tc

## 💻 Tech Stack
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## ✨ Features
- Search doctors by specialization
- Display doctor details
- Simple and user-friendly interface
